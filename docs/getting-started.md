# 快速上手

本指南帮助你快速开始使用 OpenClaw + 淘宝桌面版 CLI。

## 前置条件

1. **安装 OpenClaw 或 CoPaw**
   - OpenClaw: TypeScript/Node.js 版本
   - CoPaw: Python 版本
   
2. **安装淘宝桌面版 CLI**
   ```bash
   # 安装淘宝桌面版 Skill
   copaw skills install taobao-native
   ```

3. **安装淘宝桌面版客户端**
   - 从淘宝官网下载并安装
   - 确保客户端已登录

4. **配置 IM Channel**
   - 钉钉：推荐，功能最完善
   - 微信：需额外配置
   - Telegram：适合海外用户

## 五分钟快速体验

### 步骤 1：启动助手

```bash
# 启动 CoPaw
copaw app
```

### 步骤 2：在 IM 中发送消息

打开钉钉/微信，给你的机器人发送：

```
帮我搜索一下 iPhone 15 手机壳
```

### 步骤 3：体验 AI 购物助手

AI 助手会自动：
1. 打开淘宝桌面版
2. 搜索商品
3. 返回搜索结果

## 案例「三件套」

每个案例包含以下内容：

### 1. 场景描述
介绍这个案例解决什么问题，适合什么用户。

### 2. 对话逻辑
说明 AI 助手如何理解和处理用户请求，关键步骤是什么。

### 3. 提示词
可直接复制使用的提示词，格式如下：

```
帮我监控这个商品的价格：
商品链接：[你的商品链接]
目标价格：[你的目标价格]
```

### 4. 效果演示
模拟对话形式，展示交互过程：

::: info 用户
帮我监控这款手机壳，降价到 20 元以下通知我
:::

::: tip 助手
好的，我已经帮你设置价格监控：
- 商品：iPhone 15 透明手机壳
- 当前价格：¥29.9
- 目标价格：¥20
- 监控状态：已开启

降价后我会第一时间通知你！
:::

## 进阶用法

### 自定义提示词

你可以根据自己的需求修改提示词，比如：

```
帮我监控这个商品，如果降到 [价格] 以下，就自动加入购物车
```

### 组合多个案例

把多个案例组合起来，实现更复杂的场景：

```
帮我整理本月的消费记录，按类别统计，
然后找出可以省钱的地方，推荐更便宜的替代商品
```

## 常见问题

### Q: 淘宝桌面版没有反应？

检查：
1. 淘宝桌面版是否已登录
2. CLI 是否正确安装
3. 查看助手日志

### Q: IM 消息没有响应？

检查：
1. Channel 是否正确配置
2. 机器人是否在线
3. 消息格式是否正确

### Q: 如何查看日志？

```bash
copaw daemon logs
```
