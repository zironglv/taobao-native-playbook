# 囤货提醒

## 场景描述

家里的日用品用完了才发现没囤？卫生纸、洗衣液、牙膏... 总是等到用完了才想起要买。

**囤货提醒**帮你监控常用物品库存，快用完时自动提醒你补货，不用再担心断货。

**适合用户**：
- 总是忘记囤货的人
- 家庭主妇/主夫
- 想省心管理家庭日用品

## 对话逻辑

```
用户设置囤货清单
       ↓
AI 记录商品和提醒周期
       ↓
定时检查是否需要补货
       ↓
库存不足时发送提醒
       ↓
用户一键下单
```

**关键步骤**：
1. **清单建立**：录入常用囤货品
2. **周期设置**：设置每件商品的补货周期
3. **定时提醒**：按周期自动提醒
4. **快速下单**：一键购买常用商品

## 提示词

**基础版**：

```
帮我设置囤货提醒：
- 卫生纸
- 洗衣液
- 牙膏
- 沐浴露
```

**详细版**：

```
帮我管理家庭囤货清单：

商品清单：
1. 卫生纸 - 大约 1 个月用完
2. 洗衣液 - 大约 2 个月用完
3. 牙膏 - 大约 1.5 个月用完
4. 沐浴露 - 大约 2 个月用完

请在库存不足时提醒我补货！
```

**手动记录版**：

```
我刚买了这些东西，帮我记一下：
- 卫生纸 12卷装
- 洗衣液 2kg
- 牙膏 2支

下次需要囤货的时候提醒我。
```

## 小贴士

- 💡 **记录购买时间**：买了就告诉助手，方便计算周期
- 💡 **批量囤货**：可以一次囤多件，省运费
- 💡 **调整周期**：实际用量和预估不同时，及时调整
