# 智能补货

## 场景描述

囤货提醒告诉你该买了，但不想一个个搜索、比价、下单？太麻烦了。

**智能补货**帮你一键完成补货流程：自动找之前买过的商品，检查价格变化，有优惠就下单，没优惠也帮你加购。

**适合用户**：
- 想快速补货的人
- 不想重复搜索比价
- 囤货清单比较多

## 对话逻辑

```
用户请求补货
       ↓
AI 获取囤货清单
       ↓
搜索上次购买的商品
       ↓
检查价格和优惠
       ↓
智能推荐下单方案
```

**关键步骤**：
1. **清单获取**：读取用户的囤货清单
2. **历史查询**：找出上次购买的商品
3. **价格检查**：对比当前价格和历史价格
4. **智能决策**：便宜就买，贵就等

## 提示词

**基础版**：

```
帮我补货囤货清单里的东西
```

**指定商品版**：

```
帮我补货以下商品：
1. 卫生纸
2. 洗衣液
3. 牙膏

直接买我之前买过的那款。
```

**智能版**：

```
帮我智能补货：

1. 检查囤货清单中需要补货的商品
2. 找到我之前买过的款式
3. 对比价格，降价优先买
4. 没降价的也帮我加购

告诉我总金额，我来确认下单。
```
