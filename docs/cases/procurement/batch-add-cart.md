# 表格批量加购

## 场景描述

公司采购，给你一张 Excel 表格，里面有几十上百个商品要买。一个一个搜索、加购，费时费力，还容易出错。

**表格批量加购**帮你导入采购表格，自动搜索商品，一键批量加购，大幅提升采购效率。

**适合用户**：
- 企业采购人员
- 需要批量采购的用户
- 有采购表格的批量买家

## 对话逻辑

```
用户上传采购表格
       ↓
AI 解析表格内容
       ↓
搜索匹配商品
       ↓
用户确认商品匹配
       ↓
批量加入购物车
```

**关键步骤**：
1. **表格解析**：读取 Excel/CSV 中的商品信息
2. **商品搜索**：根据商品名称/SKU 搜索
3. **匹配确认**：用户确认搜索结果是否正确
4. **批量加购**：一键加入购物车

## 提示词

**基础版**：

```
帮我导入采购表格并加购：
[上传表格文件]
```

**指定列版**：

```
帮我处理这个采购表格：

表格列说明：
- A列：商品名称
- B列：规格/型号
- C列：数量
- D列：预算价格（可选）

请帮我搜索并加购。
```

**带链接版**：

```
帮我批量加购这个表格里的商品：

表格里有商品链接列，直接用链接加购即可。
数量按表格里的数量列来。

完成后告诉我加购结果。
```

## 小贴士

- 💡 **表格规范**：商品名称尽量完整，方便匹配
- 💡 **同店优先**：尽量选同一店铺，省运费
- 💡 **保存记录**：导出采购记录方便报销
