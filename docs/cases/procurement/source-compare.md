# 寻源比价

## 场景描述

公司需要采购某种物料，但不知道哪里有卖、哪家便宜、哪家质量好。需要花大量时间去搜索、对比。

**寻源比价**帮你自动搜索货源，对比多个供应商的价格、评分、资质，快速找到最优选择。

**适合用户**：
- 企业采购人员
- 需要寻找货源的用户
- 需要多供应商对比

## 对话逻辑

```
用户描述采购需求
       ↓
AI 搜索相关商品
       ↓
筛选符合条件的供应商
       ↓
对比价格、评分、资质
       ↓
生成对比报告
```

**关键步骤**：
1. **需求理解**：理解采购的商品规格要求
2. **货源搜索**：搜索符合条件的商品
3. **供应商筛选**：根据评分、销量筛选
4. **对比报告**：生成可视化对比表

## 提示词

**基础版**：

```
帮我找这个商品的货源并比价：
[商品名称/描述]
```

**详细版**：

```
帮我寻源比价：

采购需求：
- 商品：[商品名称]
- 规格：[详细规格]
- 数量：[数量]
- 预算：[预算范围]
- 要求：[特殊要求，如：天猫店、销量1000+、评分4.8+]

请给我 3-5 个供应商选项并对比。
```

**更新表格版**：

```
帮我寻源并更新表格：

采购表格：[上传表格]
需求：根据表格里的商品名称找货源

要求：
1. 每个商品找 3 个供应商
2. 对比价格、评分、发货地
3. 推荐最优供应商
4. 更新表格，填入供应商信息
```

## 小贴士

- 💡 **多方比价**：至少对比 3 家供应商

- 💡 **确认发票**：提前确认能否开票、开什么票
