# 供应商管理

## 场景描述

采购需要长期合作的供应商，但供应商信息分散在各个订单、聊天记录里。想对比供应商表现，又不知道从何入手。

**供应商管理**帮你整合供应商信息，跟踪采购历史，分析供应商表现，建立长期合作关系。

**适合用户**：
- 企业采购人员
- 需要管理多个供应商
- 想建立供应商档案

## 对话逻辑

```
用户请求供应商管理
       ↓
AI 分析历史订单提取供应商
       ↓
统计各供应商采购记录
       ↓
分析价格、质量、服务表现
       ↓
生成供应商档案和评估报告
```

**关键步骤**：
1. **数据提取**：从订单中提取供应商信息
2. **档案建立**：建立供应商档案库
3. **表现分析**：分析价格、质量、响应速度
4. **评级建议**：给出供应商评级和建议

## 提示词

**基础版**：

```
帮我整理一下我买过的供应商
```

**详细版**：

```
帮我建立供应商档案：

1. 从历史订单中提取供应商信息
2. 统计每个供应商的采购金额、次数
3. 分析价格优势、发货速度、服务质量
4. 给供应商评级（A/B/C/D）

生成一份供应商管理报告。
```

**对比版**：

```
帮我对比这几个供应商：

供应商A：[店铺名/链接]
供应商B：[店铺名/链接]
供应商C：[店铺名/链接]

对比维度：价格、评分、发货速度、开票能力
```

## 小贴士

- 💡 **建立档案**：每次采购后更新供应商档案
- 💡 **定期评估**：每季度评估一次供应商表现
- 💡 **分散风险**：每个品类至少 2 家供应商
