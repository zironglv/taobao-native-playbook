# 凑单计算器

## 场景描述

满 200 减 30，满 500 减 80，满 1000 减 150... 各种满减活动让人眼花缭乱。想凑单又不知道怎么凑最划算？凑多了觉得亏，凑少了享受不到优惠。

**凑单计算器**帮你自动计算最优凑单方案，既享受最大优惠，又不花冤枉钱。

**适合用户**：
- 大促期间凑单的省钱达人
- 想享受满减但不想买多余东西
- 数学不好的"凑单困难户"

## 对话逻辑

```
用户输入当前购物车金额和满减规则
       ↓
AI 计算距离下一档优惠的差额
       ↓
搜索凑单商品池（低价、实用）
       ↓
智能推荐凑单商品组合
       ↓
计算最终优惠和实付金额
```

**关键步骤**：
1. **金额分析**：计算当前金额和优惠档位的差距
2. **凑单品搜索**：搜索低价、实用的凑单候选
3. **组合优化**：找出最优凑单组合
4. **性价比计算**：计算凑单是否划算

## 提示词

**基础版**：

```
帮我凑单：
当前金额：[输入金额]
满减规则：满[金额1]减[优惠1]，满[金额2]减[优惠2]
```

**详细版**：

```
帮我计算最优凑单方案：

当前购物车：¥[金额]
可享优惠：满 200 减 30

请告诉我：
1. 还差多少可以享受满减
2. 推荐的凑单商品（价格区间 ¥10-50）
3. 凑单后最终实付金额
4. 凑单是否划算（优惠是否大于凑单品价格）
```

**多档满减版**：

```
大促凑单求助！

当前购物车：
- 商品A：¥168
- 商品B：¥89
- 商品C：¥45
合计：¥302

满减规则：
- 满 300 减 50
- 满 500 减 100
- 满 1000 减 200

帮我算算怎么凑最划算！
```

## 小贴士

- 💡 **凑单品原则**：选刚需品，不要为凑单买无用品
- 💡 **计算性价比**：优惠 > 凑单品价格 才划算
- 💡 **跨店凑单**：多店满减可以跨店凑单
