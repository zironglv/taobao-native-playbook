# 购物车智能清结算

## 场景描述

购物车加了一堆东西，每次看到都想买，但又不想一次性清空？或者购物车里有些商品已经下架、涨价，需要清理？

**购物车智能清结算**帮你一键处理购物车，自动筛选有效商品，清理失效链接，计算最优优惠，让你快速完成结算。

**适合用户**：
- 购物车永远清不完的"囤积症"用户
- 想快速结算不想逐个检查
- 想了解购物车整体优惠情况

## 对话逻辑

```
用户请求清理购物车
       ↓
AI 调用淘宝 CLI 获取购物车
       ↓
分析购物车商品状态
       ↓
识别失效/涨价/缺货商品
       ↓
计算可用优惠券/满减
       ↓
返回清理建议和结算方案
```

**关键步骤**：
1. **获取购物车**：调用 CLI 读取购物车数据
2. **状态检查**：检查商品是否下架、涨价、缺货
3. **优惠计算**：计算最优优惠组合
4. **智能建议**：给出清理和结算建议


## 小贴士

- 💡 **定期清理**：购物车商品超过 30 个就该清理了
- 💡 **关注降价**：加入购物车后降价是捡漏好时机
- 💡 **凑单技巧**：差一点满减时看看有没有低价凑单品
