# 智能比价

## 场景描述

同样的商品，不同店铺价格可能相差很大。手动比价需要打开多个页面，一件件对比，费时费力。

**智能比价**让你只需发送商品名称或图片，AI 助手就会自动搜索全平台，找出同款商品，按价格排序，帮你找到最划算的选择。

**适合用户**：
- 追求性价比的精明买家
- 不想花时间比价的懒人
- 担心买贵的纠结症患者

## 对话逻辑

```
用户发送商品名称/图片/链接
       ↓
AI 识别商品关键信息
       ↓
调用淘宝 CLI 搜索同款
       ↓
筛选有效结果
       ↓
按价格/评分/销量排序
       ↓
返回比价结果供用户选择
```

**关键步骤**：
1. **商品识别**：从用户输入中提取商品关键词
2. **同款搜索**：使用图片搜索或关键词搜索找同款
3. **数据清洗**：过滤无效链接、异常价格
4. **智能排序**：综合考虑价格、评分、销量

## 提示词

**基础版**：

```
帮我比价：[商品名称或描述]

要求：
1. 找出 5 个同款商品
2. 按价格从低到高排序
3. 显示店铺名称、价格、评分、销量
```

**图片比价版**：

```
帮我找这个商品的最便宜同款：
[发送商品图片]
```

**精准比价版**：

```
帮我对比这款商品在不同店铺的价格：

目标商品：[商品名称]
规格要求：[尺寸/颜色/版本]
预算范围：[最低价] - [最高价]
店铺要求：[天猫优先/销量1000+/评分4.8+]

请给我一个比价表格，并推荐最划算的选择。
```

## 小贴士

- 💡 **看评分更要看差评**：高分店也可能有坑
- 💡 **销量不能只看数字**：刷单店铺销量虚高
- 💡 **发货地有讲究**：产业集群地发货更靠谱
