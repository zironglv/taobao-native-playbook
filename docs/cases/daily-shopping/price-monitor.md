# 价格监控提醒

## 场景描述

你是否遇到过这种情况：看中了一件商品，但觉得价格有点贵，想着"等降价再买"，结果忘了这件事，等想起来时发现已经恢复原价了？

**价格监控提醒**可以帮你解决这个问题。你只需要告诉 AI 助手你要监控的商品和目标价格，它就会持续监控商品价格变化，一旦降到目标价格，立即通知你。

**适合用户**：
- 等打折的"等等党"
- 想省钱的精打细算型用户
- 有明确心理价位的理性消费者

## 对话逻辑

```
用户发送监控请求
       ↓
AI 解析商品链接和目标价格
       ↓
调用淘宝 CLI 获取当前商品信息
       ↓
设置定时任务监控价格
       ↓
价格变动时检查是否达到目标
       ↓
达到目标 → 发送通知
未达到 → 继续监控
```

**关键步骤**：
1. **解析商品链接**：从用户消息中提取淘宝商品链接
2. **获取商品信息**：调用 CLI 获取商品名称、当前价格、店铺信息
3. **创建监控任务**：设置定时任务，定期检查价格
4. **价格变动通知**：价格达到目标时通过 IM 通知用户

## 提示词

复制以下提示词，发送给你的 AI 助手：

```
帮我设置一个价格监控：

商品链接：[粘贴你的淘宝商品链接]
目标价格：[输入你的目标价格，如 99 元]
通知方式：降价后立即通知我

请告诉我：
1. 当前商品价格是多少
2. 需要降价多少才能达到目标
3. 监控已设置成功
```

**进阶版提示词**（支持多商品监控）：

```
帮我监控以下商品的价格，降到目标价就通知我：

1. 商品：[链接1]
   目标价：[价格1]
   
2. 商品：[链接2]
   目标价：[价格2]

3. 商品：[链接3]
   目标价：[价格3]

请用表格形式告诉我当前价格和需要降价的幅度。
```

## 小贴士

- 💡 **设置合理目标价**：参考历史最低价，设置可达成的目标
- 💡 **批量监控**：把想买的商品都加进去，坐等降价通知
- 💡 **大促期间**：618、双11期间可设置更激进的目标价
