# 逛街参谋

## 场景描述

逛街看到喜欢的衣服，不知道值不值得买？价格贵不贵？适不适合自己？

**逛街参谋**帮你实时分析商品，告诉你价格是否合理、是否适合你的风格、和衣橱里的衣服能不能搭。

**适合用户**：
- 购物容易冲动的人
- 想买得值的人
- 不知道衣服适不适合自己

## 对话逻辑

```
用户发送商品链接/图片
       ↓
AI 获取商品信息
       ↓
分析价格、风格、质量
       ↓
匹配用户衣橱风格
       ↓
给出购买建议
```

**关键步骤**：
1. **商品分析**：价格、风格、评价
2. **价格对比**：对比同款价格
3. **风格匹配**：分析是否适合用户
4. **搭配建议**：和现有衣服能否搭配

## 提示词

