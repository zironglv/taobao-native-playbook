# 每日穿搭推荐

## 场景描述

每天早上醒来，站在衣柜前发呆：今天穿什么？天气热还是冷？要见什么人？

**每日穿搭推荐**帮你根据天气、日程、场合，智能推荐今天的穿搭，让你不用再为穿什么发愁。

**适合用户**：
- 每天不知道穿什么的人
- 想尝试新搭配的人
- 追求效率的上班族

## 对话逻辑

```
用户请求今日穿搭
       ↓
AI 获取天气、日程信息
       ↓
匹配衣橱中的合适单品
       ↓
生成穿搭方案
       ↓
展示搭配效果
```

**关键步骤**：
1. **信息收集**：天气、温度、日程
2. **单品筛选**：从衣橱选择合适的衣服
3. **搭配生成**：组合成完整穿搭
4. **场景适配**：考虑场合和活动

## 提示词

**基础版**：

```
今天穿什么？
```

**详细版**：

```
帮我推荐今天的穿搭：

天气：[天气，如：晴，15-22度]
场合：[场合，如：上班 + 晚上有约会]
风格：[风格，如：商务休闲]

推荐 2-3 套搭配方案。
```

**指定单品版**：

```
我新买了一件牛仔外套，帮我看看怎么搭：

单品：牛仔外套
风格：休闲
场合：周末出门

告诉我可以搭我衣橱里的哪些衣服。
```

## 小贴士

- 💡 **提前准备**：前一天晚上准备好第二天衣服
- 💡 **关注天气**：温度变化影响穿衣选择
- 💡 **场合意识**：不同场合穿不同风格
