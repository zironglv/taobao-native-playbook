# 衣橱数字化

## 场景描述

衣柜里衣服很多，但总感觉没衣服穿？不知道自己有什么衣服，经常买重复的款式？

**衣橱数字化**帮你把网购的衣服自动整理成电子衣橱，方便查看、穿搭和管理。

**适合用户**：
- 衣服多但不知道怎么搭的人
- 想管理自己衣橱的人
- 经常买重复款式的人

## 对话逻辑

```
用户请求导入衣橱
       ↓
AI 调用淘宝 CLI 获取服装类订单
       ↓
提取商品图片、品牌、类别、颜色
       ↓
生成电子衣橱
       ↓
支持搜索、筛选、搭配
```

**关键步骤**：
1. **订单筛选**：筛选服装类订单
2. **信息提取**：提取商品的关键属性
3. **分类整理**：按类别/颜色/季节分类
4. **穿搭建议**：基于衣橱内容推荐搭配

## 提示词

**基础版**：

```
帮我整理一下我的衣橱
```

**指定时间版**：

```
帮我导入最近一年的衣服订单，整理成衣橱：

时间范围：2023年3月 - 2024年3月
类别：上衣、裤子、外套、鞋子、配饰

请告诉我：
1. 我买了多少件衣服
2. 按类别分类统计
3. 有哪些颜色/风格
```

**新增单品版**：

```
帮我把这单加到衣橱里：
订单号：[订单号]
类别：[类别]
```
