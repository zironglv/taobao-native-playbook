# 购物习惯洞察

## 场景描述

你是一个什么样的消费者？冲动型？囤货型？价格敏感型？了解自己的购物习惯，才能更好地控制消费。

**购物习惯洞察**帮你分析购物行为，了解自己的消费风格，发现可以改进的地方。

**适合用户**：
- 想了解自己消费习惯的人
- 想改善购物习惯的人
- 想更好地控制预算的人

## 对话逻辑

```
用户请求消费分析
       ↓
AI 获取历史订单数据
       ↓
分析购物行为模式
       ↓
识别消费特征
       ↓
生成画像和建议
```

**关键步骤**：
1. **数据收集**：读取历史订单
2. **行为分析**：购买时间、频率、价格敏感度
3. **特征识别**：识别用户消费类型
4. **建议生成**：针对性改进建议

## 提示词

**基础版**：

```
帮我分析一下我的购物习惯
```

**详细版**：

```
帮我分析我的消费习惯：

时间范围：最近 6 个月

请告诉我：
1. 我是什么类型的消费者
2. 购物时间偏好
3. 价格敏感度
4. 冲动消费情况
5. 可以改进的地方
```

**深度分析版**：

```
帮我做个消费者画像：

分析维度：
- 购物频率
- 价格敏感度
- 类别偏好
- 品牌偏好
- 决策风格

给我一份详细的消费者画像报告。
```

**📊 行为洞察**

**购物时间偏好**
| 时间段 | 下单比例 |
|--------|----------|
| 晚上 20-23 点 | 45% |
| 中午 12-14 点 | 25% |
| 其他时间 | 30% |

**价格决策风格**
- 50% 的商品购买前对比过价格
- 30% 的商品使用了优惠券
- 20% 的商品直接下单

**类别偏好**
- 📱 数码电子：你的最爱，占 35%
- 👔 服饰鞋包：第二选择，占 25%
- 🏠 家居日用：囤货首选，占 20%

**⚠️ 发现的问题**

1. **囤货过多**
   - 有 12 件商品买后没使用
   - 建议买前想想是否真的需要

2. **晚上冲动下单**
   - 45% 的订单在晚上下单
   - 建议放入购物车冷静一晚

3. **优惠使用不足**
   - 只有 30% 使用了优惠券
   - 建议下单前先搜优惠

**💡 改进建议**

| 问题 | 建议 |
|------|------|
| 囤货过多 | 买前问自己：一周内会用吗？ |
| 晚上冲动 | 放入购物车，第二天再看 |
| 优惠不足 | 关注店铺优惠券、平台活动 |

**🏆 你的优点**
- ✅ 冲动消费少，大部分是计划内购买
- ✅ 会对比价格，不是盲买型
- ✅ 品牌选择理性，不盲目追品牌

继续保持理性消费！👍

## 小贴士

- 💡 **了解自己**：知道习惯才能改习惯
- 💡 **设定规则**：比如"晚上不买超过 ¥100 的东西"
