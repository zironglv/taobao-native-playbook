# 月度消费账单

## 场景描述

每个月花多少钱网购？钱都花在哪了？哪里可以省钱？

**月度消费账单**帮你自动统计淘宝消费，按类别、店铺、时间分析，让你清楚知道钱去哪了。

**适合用户**：
- 想了解消费习惯的人
- 需要记账的用户
- 想省钱的人

## 对话逻辑

```
用户请求消费账单
       ↓
AI 调用淘宝 CLI 获取订单数据
       ↓
按类别/店铺/时间分类统计
       ↓
生成可视化报告
       ↓
提供省钱建议
```

**关键步骤**：
1. **数据获取**：读取订单数据
2. **分类统计**：按类别/店铺归类
3. **趋势分析**：对比往月数据
4. **洞察建议**：发现省钱机会

## 提示词

**基础版**：

```
帮我看看上个月花了多少钱
```

**详细版**：

```
帮我生成 3 月的消费账单：

请告诉我：
1. 总消费金额
2. 按类别分布
3. 消费最多的店铺 TOP 5
4. 和上月对比
5. 省钱建议
```

**指定时间版**：

```
帮我统计 [时间范围] 的消费：

时间范围：2024年1月1日 - 2024年3月31日
统计维度：按月份、按类别、按店铺

生成一份季度消费报告。
```
