# 节日送礼攻略

## 场景描述

母亲节、父亲节、情人节、圣诞节... 各种节日让人应接不暇。每次到了节日就开始头疼：送什么好？买什么合适？

**节日送礼攻略**帮你提前规划节日礼物，根据不同节日和收礼人，推荐最合适的礼物方案。

**适合用户**：
- 每逢节日就头疼的"送礼困难户"
- 想在节日送出心意的人
- 需要批量准备节日礼物

## 对话逻辑

```
用户询问节日礼物
       ↓
AI 识别节日类型和收礼人
       ↓
根据节日特点推荐礼物
       ↓
搜索商品并对比
       ↓
提供购买建议
```

**关键步骤**：
1. **节日识别**：判断是什么节日
2. **关系匹配**：针对不同收礼人推荐
3. **场景适配**：考虑节日氛围和习俗
4. **预算控制**：提供不同价位选择

## 提示词

**基础版**：

```
[节日名称]快到了，帮我推荐礼物

收礼人：[关系，如：妈妈/爸爸/对象/朋友]
预算：[金额]
要求：[特殊要求]
```

**多收礼人版**：

```
春节快到了，帮我准备礼物清单：

需要送礼的人：
1. 妈妈 - 预算 500 左右
2. 爸爸 - 预算 500 左右
3. 丈母娘 - 预算 800 左右
4. 老丈人 - 预算 800 左右

每人推荐 2-3 个选项。
```

**节日日历版**：

```
帮我列一下接下来一年的重要节日，以及每个节日的送礼建议。
```
