# 生日礼物管家

## 场景描述

家人生日快到了，想送礼物但不知道送什么好？要么忘了日子，要么临时抓瞎乱买，送的东西对方不一定喜欢。

**生日礼物管家**帮你记住重要日子，提前提醒，根据收礼人特点推荐合适的礼物，还能直接下单购买。

**适合用户**：
- 经常忘记家人生日的人
- 不知道送什么礼物的人
- 想送有意义礼物的人

## 对话逻辑

```
用户录入生日信息
       ↓
AI 分析收礼人特点（年龄/性别/爱好）
       ↓
根据关系推荐礼物类型
       ↓
搜索相关商品并比价
       ↓
用户选择后帮下单
```

**关键步骤**：
1. **信息收集**：录入生日、关系、喜好
2. **智能提醒**：生日前 7 天提醒
3. **礼物推荐**：根据收礼人画像推荐
4. **下单购买**：一键下单配送

## 提示词

**录入生日版**：

```
帮我记住这些生日：

1. 老婆：3月17日
2. 大侄子：3月18日（小学1年级，喜欢奥特曼）
3. 大姑丈：3月26日

生日前一周提醒我准备礼物！
```

**礼物推荐版**：

```
我老婆快过生日了，帮我推荐礼物：

- 关系：老婆
- 年龄：30岁左右
- 喜好：喜欢护肤、追剧、偶尔化妆
- 预算：1000元左右

推荐几个合适的礼物选项。
```

**一键送礼版**：

```
帮我给大侄子准备生日礼物：

- 年龄：7岁，小学1年级
- 喜欢：奥特曼
- 预算：100-200元
- 要求：要有趣、适合男孩

推荐并帮我下单，地址用我默认地址。
```
