# 批量开发票

## 场景描述

公司报销需要发票，之前网购的订单一个个去开发票太麻烦。有的订单忘记开，有的开错了，报销时各种问题。

**批量开发票**帮你一键为所有订单申请发票，自动整理发票信息，方便报销。

**适合用户**：
- 需要公司报销的上班族
- 经常网购的开发票用户
- 想统一管理发票的人

## 对话逻辑

```
用户请求开发票
       ↓
AI 调用淘宝 CLI 获取订单列表
       ↓
筛选未开发票的订单
       ↓
读取发票抬头信息
       ↓
批量申请开票
       ↓
整理发票列表供下载
```

**关键步骤**：
1. **订单筛选**：找出未开发票的订单
2. **发票配置**：读取/设置发票抬头
3. **批量申请**：向各商家发送开票请求
4. **结果汇总**：整理发票下载链接

## 提示词

**基础版**：

```
帮我看看哪些订单还没开发票
```

**批量开票版**：

```
帮我批量开发票：

发票信息：
- 抬头：[公司名称]
- 税号：[税号]
- 邮箱：[接收邮箱]

要求：
1. 列出所有未开发票的订单
2. 批量申请开票
3. 发送到我的邮箱
```

**按时间开票版**：

```
帮我申请 [月份] 月份订单的发票：

时间范围：2024年3月1日 - 2024年3月31日
发票抬头：[公司名称]
税号：[税号]
邮箱：[邮箱]

请告诉我有哪些订单需要开票。
```
