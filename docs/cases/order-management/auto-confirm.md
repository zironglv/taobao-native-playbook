# 自动确认收货

## 场景描述

收到货了，忘记确认收货？系统自动确认后才发现商品有问题，维权困难。

**自动确认收货**帮你智能管理订单确认，收到货后自动检查物流状态，适时确认收货，让你不用操心。

**适合用户**：
- 购物多、订单多的用户
- 容易忘记确认收货
- 想及时确认又怕有问题

## 对话逻辑

```
AI 定期检查即将自动确认的订单
       ↓
查询物流状态（是否已签收）
       ↓
已签收 → 提醒用户确认
未签收 → 延长确认时间
       ↓
用户确认 → 自动执行确认收货
```

**关键步骤**：
1. **监控订单**：定期扫描即将自动确认的订单
2. **物流检查**：确认用户是否已收货
3. **智能提醒**：在合适的时机提醒用户
4. **自动执行**：用户同意后自动确认

## 提示词

**基础版**：

```
帮我检查有哪些订单需要确认收货
```

**自动管理版**：

```
帮我管理订单确认收货：

1. 每天检查即将自动确认的订单
2. 已签收且没问题 → 提醒我确认
3. 未签收 → 延长确认时间
4. 我回复"确认"后自动执行

可以吗？
```

**批量确认版**：

```
帮我确认这些订单的收货：
[订单号1]
[订单号2]
[订单号3]
```

## 小贴士

- 💡 **验货再确认**：确认收货后维权更难
- 💡 **关注时间**：自动确认前 3 天系统会提醒
- 💡 **延长确认**：有问题可以申请延长确认时间
