# 物流追踪汇总

## 场景描述

买了好多东西，分散在不同快递公司，要打开多个 App 查物流？太麻烦了！

**物流追踪汇总**帮你一键查看所有包裹状态，不用再一个个 App 切换。

**适合用户**：
- 购物多的"快递大户"
- 想快速知道包裹到哪了
- 不想装多个快递 App

## 对话逻辑

```
用户请求查看物流
       ↓
AI 调用淘宝 CLI 获取订单列表
       ↓
筛选已发货未签收的订单
       ↓
查询各快递物流信息
       ↓
按状态分类汇总
```

**关键步骤**：
1. **获取订单**：读取所有进行中的订单
2. **查询物流**：调用快递 API 获取物流信息
3. **状态分类**：按配送中/已到站/派送中等分类
4. **智能提醒**：即将送达的优先显示

## 提示词

**基础版**：

```
帮我看看包裹到哪了
```

**详细版**：

```
帮我汇总所有包裹的物流状态：

1. 按状态分类（运输中/已到站/派送中/已签收）
2. 标注即将送达的包裹
3. 显示每个包裹的最新轨迹

谢谢！
```

**指定订单版**：

```
帮我追踪这几个订单的物流：
订单号：[订单号1], [订单号2], [订单号3]
```
