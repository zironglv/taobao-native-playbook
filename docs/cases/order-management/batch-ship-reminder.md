# 批量催发货

## 场景描述

买了好多东西，商家迟迟不发货？一个一个去催太麻烦，还可能漏掉。

**批量催发货**帮你一键检查所有待发货订单，自动催促商家发货，让你不用再一个个点。

**适合用户**：
- 购物多、订单多的用户
- 着急收货的用户
- 不想手动催发货的懒人

## 对话逻辑

```
用户请求催发货
       ↓
AI 调用淘宝 CLI 获取待发货订单
       ↓
筛选超过 N 小时未发货的订单
       ↓
批量发送催发货消息
       ↓
返回催发货结果汇总
```

**关键步骤**：
1. **获取订单列表**：读取待发货订单
2. **时间筛选**：找出超时未发货的订单
3. **批量催发**：向商家发送催发货消息
4. **结果汇总**：统计催发货成功率

## 提示词

**基础版**：

```
帮我催一下还没发货的订单
```

**指定时间版**：

```
帮我催发货：

筛选条件：超过 [24] 小时未发货的订单
催发货话术：麻烦尽快发货，谢谢！

请告诉我催发货结果。
```

**批量版**：

```
检查我所有待发货的订单，帮我：
1. 列出所有未发货订单
2. 标注超过 24 小时未发货的
3. 批量发送催发货消息
4. 汇总催发货结果
```

## 小贴士

- 💡 **催发货时机**：超过 24 小时催一次，超过 48 小时可以申请退款
- 💡 **话术建议**：礼貌催促效果更好
- 💡 **保留证据**：长时间不发货可截图维权
